Description

This widget displays a rating (0-5 stars) and allows the user to alter the rating by clicking.

Property Details

The "rating" property

  "rating"                   gdouble               : Read / Write

The rating displayed in the widget, as a floating point value between 0.0 and 5.0.

Allowed values: [0,5]

Default value: 2.5
